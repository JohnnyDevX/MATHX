Zbiory Liczb

Zbiór liczb rzeczywistych zawiera zbiór liczb naturalnych, zbiór liczb całkowitych, zbiór liczb wymiernych i zbiór liczb niewymiernych.

N: { 0, 1, 2, 3, ...}

C: { ..., -2, -1, 0, 1, 2, ... }

W: { ..., -1/2, 0, 1, 5/4, ... }

NW: { ..., -√3, -√2, √2, √3, ... }

Liczby naturalne
Liczby od 0 do nieskończoności w których każda kolejna jest o 1 większa od poprzedniej.
